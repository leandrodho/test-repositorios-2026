| *Sistema de Buscador Unificado de Recursos para Bibliotecas Físicas y Virtuales NexusLib, Universidad Privada de Tacna – 2026.* | |
| | |
| **Propósito del Proyecto y Resultados esperados:** <br><br> El propósito de este proyecto es optimizar la localización y el acceso a recursos bibliográficos híbridos de la universidad, unificando de manera simultánea y concurrente las consultas del catálogo físico local con repositorios digitales externos (Alpha Cloud y E-Libro) bajo una plataforma web centralizada y de alta disponibilidad. <br><br> Los resultados esperados son: <br> • Reducción drástica en los tiempos de respuesta y localización exacta (piso y estante) de recursos bibliográficos en sala. <br> • Incremento definitivo en la tasa de uso, descargas y visibilidad de las colecciones académicas virtuales ya financiadas por la institución. <br> • Sincronización automatizada en tiempo real del stock de inventario y del seguimiento transaccional de solicitudes de reservas y libros guardados. | |
| | |
| **Población Objetivo:** <br><br> • Estudiante / Investigador. <br> • Docente. <br> • Personal Bibliotecario / Administrador. | |
